Nuestras intenciones

Con este proyecto para el curso de plataformas de programación empresarial se pretende construir el front-end de un sitio web de una temática arbitraria a partir de una plantilla, con base en ella aplicaremos algunas cosas vistas previamente en el curso, luego diseñaremos el back-end del sitio web y así Creature un portal funcional en su totalidad.
En este sitio web, nosotros hemos escogido la temática del skateboarding, ya que nos llama la atención a los dos y trataremos de dar buena información y coherencia al sitio web.

Repositorio

Tiene mínimo 30 commits en fechas distribuídas entre el inicio del ejercicio y la entrega final.

Registro

Hay registro de usuarios e inicio de sesión contra el localstorage.

Funcionalidades

Se agregan las funcionalidades requeridas en el home (rotador de banners, menú fijo...)

Gulp

Se usa gulp y se minifican las hojas de estilo y el código de javascript.

Navegación

Se puede navegar correctamente y se realiza la división de secciones en diferentes páginas.

API

Se consume una API y se controla la visualización de la sección solamente bajo logeo.
